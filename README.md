# SQL Case Studies

A portfolio of SQL and database design case studies.

## Projects

### 1. Business Travel Management
Relational model, ER diagram, DDL, seed data, and analytical SQL queries for a business travel domain.

### 2. Tabletop App
Database design for a board game / tabletop RPG application with users, friendships, personal game collections, events, and registrations.

## Repository structure

- `business-travel/` — case study on business travel management
- `tabletop-app/` — case study on a tabletop gaming application

Each project contains:

- `sql/` — ddl script, seed data, and query scripts
- `report/` — report in html format
